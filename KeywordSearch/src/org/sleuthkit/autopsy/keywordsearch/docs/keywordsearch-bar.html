Keyword Search Bar

The keyword search bar is used to search the index for matching words, phrases, lists, or regular expressions. It can also be used during ingest to add images to the search process. Enable regular expression mode by pressing the arrow to the left of the search box and selecting 'Use Regular Expressions'

Keyword List Search

Lists created using the Keyword Search Configuration Dialog can be searched by pressing on the 'Keyword Lists' button, selecting the check boxes corresponding to the lists to be searched, and pressing the 'Search' button.

Search During Ingest

Searching during ingest is not supported. However, lists can be added to ingest by following the same procedure as above.
